# 🐍 Python Trainer

**Образовательный веб-тренажёр для проверки знаний Python.**

Приложение позволяет пользователям проходить тесты по различным темам (синтаксис, циклы, ООП и др.), видеть результат мгновенно и отслеживать свой прогресс в личном кабинете.

---

## Функционал

*   **Авторизация и Регистрация:** Система учета пользователей.
*   **Квиз-система:** Решение задач с валидацией ответов.
*   **Уровни сложности:** Разделение заданий на легкие, средние и сложные.
*   **Личный кабинет:** Статистика успешных и неуспешных попыток.
*   **Административная панель:** Удобное управление вопросами и ответами.

---

## ﹗Установка и запуск﹗

### 1. Клонирование и окружение
Скачайте проект и создайте виртуальное окружение:

```bash
# Клонирование
git clone https://github.com/goliaf0solo/django_p_trainer.git
cd django_p_trainer

# Создание venv
python -m venv venv

# Активация (Windows)
venv\Scripts\activate

# Активация (macOS/Linux)
source venv/bin/activate
``` 

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка базы данных
Примените миграции и создайте суперпользователя (администратора):

```bash
python manage.py migrate
python manage.py createsuperuser
```

### 4. Запуск сервера

```bash
python manage.py runserver
```
Откройте браузер и перейдите по адресу: http://127.0.0.1:8000

## Наполнение заданиями (Контент)

Проект поставляется с чистой базой данных. Чтобы начать работу:

*   Перейдите в админ-панель: http://127.0.0.1:8000/admin/
*   Войдите под логином и паролем, созданным на этапе 3.
*   В разделе Core выберите Задания -> Добавить.
*   Заполните поля: Тема, Вопрос, Правильный ответ, Сложность.
*   Нажмите "Сохранить".

После этого задания появятся на главной странице.

## Проверка качества кода (Pylint)
Проект настроен для статического анализа с учётом специфики Django.
```bash
# Установка зависимостей для проверки
pip install -r requirements.txt

# Запуск анализа (игнорирует миграции и настройки)
pylint core config --ignore=migrations,settings.py
```

## Быстрое наполнение базы данных

Вместо ручного добавления через админку, можно запустить скрипт:
```bash
python seed.py
```
Скрипт безопасно добавит задания разных уровней сложности, не создавая дублей.
